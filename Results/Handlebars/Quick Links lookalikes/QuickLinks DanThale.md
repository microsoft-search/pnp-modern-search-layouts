
# Layouts for Quick Links Look-a-likes  - the Dan Thale variant#

Dan Thale added this sample to the repo :-)

Here is a layout template that I created along with AI's help that mimics the Quick Links Button format. This is super handy when you are using the same links across multiple pages. You'll need a supporting list with the following fields mapped to your respective RefinableStrings:

TargetWebPart [Choice]: This allows you to group links together for management as well as to show or hide links.
SortOrder [Number]: This allows you to specify the order that your links appear.
Title [Text]: Primary Link Title
Description [Text]: Link description text that appears below title.
URL [Text]: Link to your item/document/page etc
IconName [Text]: Microsoft Fabric UI Icon Selection
IconColor [Text]: Hex color value for the icon. Must include # as well.
Slots - You'll see in the code below that these are mapped to Slots. Please replicate per your needs.

The html is available  [Here](/Results/Handlebars/Quick%20Links%20lookalikes/QuickLinkLike_simple2.html)
